Markdown使用指南

多级标题

Markdown支持六级标题

使用方式：

一、格式：#+空格+标题

#数量指示标题级数

二、快捷键

Ctrl + 1~6 设置标题级数

Ctrl + 0 清除标题格式

有序列表

Markdown支持多级有序列表，自动调整序号

使用方式 ：

一、书写格式：序号. 标题

注意事项：

①编号后，一定要有空格！

②行末回车，序号自动添加。

③添加/删除某一行，序号自动调整

④降级快捷键：Tab

​ 升级快捷键：Shift+Tab

示例：

  1. 打开冰箱
    1. 插电
    2. 开机
    3. 开门
  2. 把大象放进冰箱
  3. 关闭冰箱

二、快捷键：选中标题后 Ctrl + Shift + [

无序列表

Markdown支持有序列表和无序列表嵌套使用

使用方式：

一、* 标题或者- 标题(注意空格）

二、快捷键：Ctrl + Shift + ]

任务列表

使用格式：- [ ] 任务

  • 任务一
  • 任务二
  • 任务三

插入表格

常用操作：

  • 调整表格行列值
  • 左中右对齐
  • 插入/删除表格行列
  • 删除表格

新建表格格式：|内容一|内容二|内容三|+回车

新建表格快捷键：Ctrl + T

注意事项：添加行时需按住Ctrl+Enter

行内代码

使用格式：内容两端添加反引号“`”(英文输入状态下ESC下的按键）

快捷键： Ctrl + Shift +`

Java 是一门面向对象的编程语言。

代码块

使用格式：三个`java 或 ~java(可替换其它语言名称)

快捷键：Ctrl + Shift + K

插入图片

快捷键：Ctrl + Shift + I
