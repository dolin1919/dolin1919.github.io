hexo建立博客

第一章

①配置环境：node

下载地址http://nodejs.cn/download/

安装过程全程默认，点击下一步即可
进入命令性窗口输入下列命令查看版本

node -v

npm -v

由于npm源下载速度较慢，替换为淘宝镜像源

npm install -g cnpm –registry=https:/registry,npm,taobao.org
等待自动安装完成后输入
cnpm
显示辅助信息证明安装成功。
②安装hexo博客框架
配置完成后使用cnpm正式安装hexo框架
输入命令
cnpm install -g hexo-cli
自动安装完成后输入命令查看版本信息
hexo -v
